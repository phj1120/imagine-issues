# 인스타그램 자동화 - ai_nim_ai

# 소개

Karlo API, Instagram Graph API 를 이용해

이용해 댓글을 수집하고 이를 기반으로 이미지를 생성해 게시하는 과정을 자동화해

관리 하지 않아도 돌아가는 시스템 제작

## 계정 소개

당신이 보고 싶은 그림을 형식에 맞게 댓글로 달아주세요.

댓글을 기반으로 이미지가 생성 됩니다.

##who [] when [] where [] what []##

## 게시글 별 소개

# 기획

1. 직전의 게시물 주제
2. 이전 게시물들에 달린 댓글 주제 (니즈 순)
3. 미리 정해둔 주제
